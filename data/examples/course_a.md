# Course A

## Description

Course A introduces Python basics and data_processing for small analytical reports.

## Topics

- Python basics
- Data processing
- Report generation

## Learning Outcomes

- Use python_basics to read a small data file.
- Apply data_processing before writing a short report.

## Competencies

- python_basics
- data_processing

## Assessments

- Mini report checks data_processing.

## Evidence

- Student submits a Markdown report with data_processing steps and conclusions.
